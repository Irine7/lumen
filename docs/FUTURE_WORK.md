# Future Work

The maintained roadmap is `docs/ROADMAP.md`.

Near-term priorities:

- Replace the deterministic development Groth16 setup with a production ceremony.
- Generate verifier constants from production verification keys.
- Move proving into a browser worker with artifact caching.
- Wire browser-submitted testnet claims through wallet/RPC integration.
- Add confidential amount support.
- Add auditor view keys for selected campaign accountability.
- Integrate real NGO/KYC providers without putting raw eligibility data on-chain.
- Add deny-list non-membership proofs.
- Generate eligibility root attestations with RISC Zero or another verifiable compute path.
- Add Freighter/mobile wallet support.
- Run external cryptography and smart contract audits.
